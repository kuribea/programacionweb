HTML

Definicion:

Lenguaje de marcado utilizado para estructurar y presentar el contenido de una página web.

CSS

Definicion:

Lenguaje utilizado para describir el aspecto y el formato de un documento HTML.

JS

Definicion:

Lenguaje de programación que permite agregar interactividad y comportamiento a las páginas web.

JSON

Definicion:

Formato de datos utilizado para el intercambio de información estructurada en aplicaciones web.

HEAD

Definicion:

Elemento en HTML que contiene información sobre el documento, como el título y los metadatos.

BODY

Definicion:

Elemento en HTML que representa el contenido principal de un documento.

SECTION

Definicion:

Elemento en HTML utilizado para agrupar contenido relacionado temáticamente.

DIV

Definicion:

Elemento en HTML utilizado para crear divisiones o contenedores para organizar y estilizar el contenido.

MODELO DOM

Definicion:

Representación en memoria de la estructura de objetos de un documento HTML, que permite la manipulación dinámica de la página.

LI

Definicion:

Elemento en HTML utilizado para representar un elemento de una lista ordenada (ol) o no ordenada (ul).

UL

Definicion:

Elemento en HTML utilizado para crear una lista no ordenada, donde los elementos se presentan en cualquier orden.

NAV

Definicion:

Elemento en HTML que representa una sección de navegación dentro de una página web.

HEADER

Definicion:

Elemento en HTML que representa el encabezado o la cabecera de una sección o del documento completo.

FOOTER

Definicion:

Elemento en HTML que representa el pie de página de una sección o del documento completo.

SPAN

Definicion:

Elemento en HTML utilizado para aplicar estilo o agrupar elementos en línea dentro de un bloque de texto.

IMG

Definicion:

Elemento en HTML utilizado para insertar imágenes en una página web.

OL

Definicion:

Elemento en HTML utilizado para crear una lista ordenada, donde los elementos se presentan en secuencia numérica.

CLASS

Definicion:

Atributo en HTML utilizado para especificar una o varias clases CSS para un elemento.

TD

Definicion:

Elemento en HTML utilizado para representar una celda en una tabla (tr).

TH

Definicion:

Elemento en HTML utilizado para representar una celda de encabezado en una tabla (tr).
